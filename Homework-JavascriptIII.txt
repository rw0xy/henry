Bucle For

EL bucle for es una sentencia que ejecuta repetidas veces un bloque de código.
Para que el bucle se detenga se tiene que cumplir una condición. en el caso de que esa condición no se cumpla, el bucle se repetira indeterminadamente. 


Operadores lógicos

Los operadores lógicos se utilizan para combinar expresiones de igualdad y para comprobar si ninguna de ellas es verdadera,si ambas con verdaderasó si sólo una de ellas es verdadera. 



&&

Este operador se denomina "and" ó "y". Está representado con este símbolo &&.

Sólo devolverá verdadero en el caso en que ambas expresiones sean veraderas. siuna de ellas es falsaoambas son falsas, devolcerá un falso.

||

Este operador se denomina "or" u "ó". está representado con el símbolo ||.

Devolverá verdadero si una de las expresiones, o ambas, es verdadera. 

si ambas expresiones son falsas, devolverá falso.

!

Este operador se llama "not" ó "no". está representado porel simbolo !. 

Devolverá el valor booleano opuesto alque se le pasa. es decir que si el valor es verdaero, devolvera falso.
